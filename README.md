# Rapid Home Service

Front-end hackathon demo for an instant home service booking platform.

## Features

- **Booking Interface**: Search, Service Selection, Time Scheduling, Payment Options.
- **Payment Processing**: Simulated payment verification and state.
- **Confirmation**: Booking summary, generated ID, WhatsApp notification simulation.
- **Design**: Premium "Startup" feel with 80-10-10 color rule.

## Stack

- React
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations)

## Running Locally

1. `npm install`
2. `npm run dev`
